# create-react-app-jzx

推荐使用 yarn

## 使用
```bash
$ git clone https://github.com/amoplayer/create-react-app-jzx.git
$ cd create-react-app-jzx
```

Install `node_modules`:

```bash
$ yarn
```

```bash
$ yarn start
```
